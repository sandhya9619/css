If the hypothesis is true and the conclusion is false, then the conditional statement is false. Likewise, if the hypothesis is false the whole statement is false.

Q.57 how many type of JS Event? How to use it ?

These include mouse events (e.g., click, dblclick, mouseover, mouseout), keyboard events (e.g., keydown, keypress, keyup), form events (e.g., submit, change, focus, blur), window events (e.g., load, resize, scroll), and document events (e.g., DOMContentLoaded). To use these events, you typically attach event listeners to HTML elements using methods like addEventListener().
